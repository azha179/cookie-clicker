# Cookie Clicker

A replica of *Cookie Clicker*, built in Python with Tkinter. This was my first ever coding project, made in 2021.

Click the cookie to bake cookies, buy buildings to generate cookies per second, and purchase upgrades to boost production. Progress can be saved and loaded.

## Running

```
python cookie_clicker.py
```

Requires Python 3 with Tkinter (included in standard installs). No third-party packages are needed. The image assets live in [`assets/`](assets/) and must stay alongside the script.

## History

The commit history walks through the eight development versions, from the first basic clicker to the final game with the full set of buildings, upgrades, and saving/loading.

The [`Archive/`](Archive/) folder preserves the original project files exactly as they were, including every numbered version and rough development notes.
